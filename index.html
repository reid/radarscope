RadarScope by reid

MyBlogLog API library for lifestreaming applications.

RadarScope leverages the aggregated social network information MyBlogLog collects and provides it in a consistent format for embedding into other applications. It was created to provide a lifestream for reidburke.com.

Dependencies

PHP 5 with allow_url_fopen and json.
